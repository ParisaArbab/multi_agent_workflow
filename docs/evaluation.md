# Evaluation Plan

This project can be evaluated with simple and useful metrics.

| Metric | What It Measures |
|---|---|
| Latency | Time needed to generate the final report |
| Report Completeness | Whether the report includes all required sections |
| Readability | Whether the report is clear and easy to understand |
| Human Evaluation | A human score for accuracy, clarity, and usefulness |
| LLM Evaluation | A second LLM checks structure, quality, and missing details |

Current status: basic evaluation plan included. Full automated scoring can be added later.
